```
- viele Fälle von Unterrichstörungen

=>  Einführung eines neuen Unterrichtstils
```

### Schritte einer Empirischen Untersuchung

1. Zielsetzung konkretisieren
2. Planung der Untersuchung / Analyse
3. Durchführung
4. Analyse
5. Schlussfolgerung



### Zielsetzung

- Geht die Anzahl der Störungen zurück
- welche Entwicklung wird prognostiziert
- gibt es einen Wochentagseffekt (vor Maßnahme / nach Maßnahme)



### Planung der Untersuchung

- Erhebung der Daten: 
    - **Beobachtung**
- erhobene Größen:
    - **Anzahl Störungen pro Tag**
    - **an aufeinanderfolgenden Schultagen**
- mit Kontrollgruppe (vor / nach)



### Planung der Analyse

welche statistischen Methoden? 

- Rückgang Anzahl Störungen  => graphische Darstellung
- Prognose => Zeitreihenanalye
- Wochentagseffekt => Zusammenhangsanalye



### Durchführung

-> im Unterricht Störungen zählen in Kontrollgruppe+ Testgruppe



### Analyse

ja keine Daten, kein Bock mitzuschreiben



### Interpretation

- und anhand Zeitreihenanalyse Zukunft **prognostizieren**
