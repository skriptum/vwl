# 03.04.2024 Institution 

## Definitionen

nach institutionalistischer Theorie:

> **Institution:** everything producing stable, valued and *recurring patterns of behaviour*

Beispiele: Heirat, warmes Mittagessen, ...



## Effekte

nach  *Lowndes / Powell*

Institutionen beeinflussen durch:

- *Regeln* (sanktioniert duch Zwangshandlungen)
- *Praktiken* (sanktioniert durch Missfallen)
- *Narrative* (sanktioniert durch UNverständnis)

=> **constraint** on behavior 



aber auch: **ermöglichen**

Beispiel: Gebühren für Wasserleitungen => niedrigere Mortalitäten aufgrund Wasserinfrastruktur

## Isomorphie / Aufbau

nach *DiMaggio / Powell*

`warum sind viele Organisationen gleichförmig?` = isomorph

Gründe für Isomorphie

- *Erzwungene* I. = Regeln erzwingen Anpassung
- *Mimetische* I. = Nachahmung von Erfolgreichen
- *Normative* I. = Beelgschaft ähnlich geprägt in Ausbildung

=> Governancestruktur als generalisierter Erzeuger von Institutionalisierungen (Staaten, Märkte, Bildungssystem, ...)



## Basisinstitution

nach *Lepsius*

> **Basisinstitution:** soziale Regulierung, durch die Anspruch auf soziale Geltung in Symbolen gezeigt wird

führt zu => legitimierter sozialer Ordnung

wie werden BasisInst. Instiutionalisiert? *Beispiel Wissenschaft*

| Schritt                                      | Beispiel = Wissenschaft        |
| -------------------------------------------- | ------------------------------ |
| Leitidee                                     | *Wissenschaftsfreiheit in GG*  |
| spezifische Kriterien für rationales Handeln | *wissenschaftliche Methoden*   |
| Isolierung eines Kontexts                    | *selbstständige Universitäten* |
| Sanktionen mit Durchsetzung                  | *schwache Sanktionen*          |

Sackmann: nicht intendierte Probleme von Institutionen => neue Institutionen



## Kognitive Institution

*nach Douglas*

> **kognitive Institution:** soziale Ordnung wird fixiert durch Verankerung im Denken von Individuen

=> Gemeinschaft mit ähnlichem Denkstil

Durch:

1. Klassifikation (bspw. Arbeitslose)
2. Wertung: (faul/ wertlos)
