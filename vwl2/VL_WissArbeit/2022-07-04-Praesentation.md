
# 04.07.2022 Praesentation

wie gestaltet man eine Rede / Referat / Vortrag? 

## Bestandteile

- Einleitung, bspw. 
    - Frage
    - Bild
    - Artikel / Geschichte
- Hauptteil
    - **strukturierter** Inhalt
- Schluss
    - Zusammenfassung
    - an Beginn anschließen
- Diskussion

## Organisation

- Karteikarten !
    - keine Sätze
    - pro Thema eine Seite
    - kurzhalten

- Folien:
    - 1-2 Minuten pro Folie
    - wenig Text
    - Abwechslungsreich
